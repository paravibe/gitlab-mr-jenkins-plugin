Adds a comment to the merge request. Blank values will be skipped.

All messages below supports environment variables.

GitLab variables

gitlabMergeRequestTitle - Title of MR

gitlabMergeRequestDescription - Description of MR

gitlabMergeRequestUrl - URL of MR

gitlabMergeRequestId - The ID of the MR

gitlabMergeRequestIid - The internal ID of the MR

gitlabMergeRequestSourceBranch - Source branch of the MR

gitlabMergeRequestTargetBranch - Target branch of the MR

gitlabMergeRequestProjectId - MR's project ID

gitlabMergeRequestProjectName - MR's project name

gitlabMergeRequestProjectWebUrl - MR's project URL

gitlabMergeRequestLastCommitId - MR's last commit

gitlabMergeRequestSshUrl - MR's ssh URL

gitlabMergeRequestWebUrl - MR's web URL

gitlabMergeRequestLabels - MR's labels

gitlabMergeRequestUserName - Name of the user that triggered a change

gitlabMergeRequestUserUsername - Username of the user that triggered a change

gitlabMergeRequestAuthorUsername - MR's author username

gitlabJobPipelineName - Pipeline name specified in the field above


Global variables can be used as well
